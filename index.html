Path Connect

Path Connect — правила

Цель

Первым построить непрерывную цепочку из своих фигур, соединяющую ваш флаг с флагом соперника. Красный соединяет 🔴→🔵, синий — 🔵→🔴.

Ход

Игроки ходят по очереди. За ход вы ставите одну фигуру на любую свободную клетку (стена занимает 3 клетки сразу). Любую пустую клетку может занять любой игрок.

Соединение

  • Свои фигуры считаются связанными, если стоят рядом по горизонтали или вертикали.
  • По диагонали связь работает только через звезду (хотя бы одна из двух соседних клеток — звезда).
  • Цепочка засчитана, когда она непрерывно тянется от одного флага до другого.

Фигуры

  • 🟢 Фигура — обычный элемент, ставится без ограничений.
  • ⭐ Звезда — 1 за партию. Даёт диагональные связи — путь может идти «наискосок».
  • 🧱 Стена — 1 за партию, занимает 3 клетки в ряд. Блокирует соперника. Клавиша R — повернуть стену. Свой путь стеной перекрывать нельзя.

Режимы

  • 👥 1 на 1 — двое за одним экраном.
  • 🧠 ИИ Perfect Engine — вы играете красным против ИИ. Обыграть его очень трудно: он просчитывает партию наперёд, защищает свой путь и не даёт себя запереть.
